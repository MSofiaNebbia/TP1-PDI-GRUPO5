## Descripcion: 

Problema 1: Ecualización local de histograma

Se implementó una función que aplica ecualización local del histograma sobre una imagen en escala de grises, considerando un vecindario de tamaño M×N centrado en cada píxel. Se analizó la influencia del tamaño de ventana en los resultados obtenidos.

Problema 2: Corrección de multiple choice

Se desarrolló un script que detecta automáticamente la estructura de un formulario de examen, identifica y clasifica las respuestas escritas  (A, B, C o D), valida los campos del encabezado (Name, Date y Class) y genera una imagen resumen con los resultados de todos los exámenes procesados.
